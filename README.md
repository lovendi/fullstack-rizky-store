# Front-end Sistem Informasi POS Toko Rizky (REACT)

git clone https://github.com/lovendi/risky-store-frontend.git

Cara menjalankan backend
1. cd /backend
2. hapus package.lock.json
3. npm install
4. npm run dev

Cara menjalankan frontend
1. cd /frontend
2. hapus package.lock.json
3. npm install
4. npm run start
